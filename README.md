# Sales_Analysis_Tableau
Project Description

This Sales Analysis Dashboard provides a comprehensive view of global sales performance across countries, markets, categories, and sub-categories. It highlights total sales, total profit, average discount, and average profit while visualizing key metrics such as country-wise sales, category performance, shipping costs, and top-selling sub-categories. The dashboard enables clear understanding of sales distribution, profit drivers, and market opportunities.
